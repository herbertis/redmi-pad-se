# 📱 Redmi Pad SE - Product Landing Page

[Preview do Projeto](https://desenvolvimento15.github.io/redmi-pad-se-MiBrasil/)

## 💻 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma **Landing Page de produto para o Redmi Pad SE**, inspirada nos padrões visuais utilizados em páginas oficiais de lançamento de produtos da Xiaomi e vitrines de e-commerce.

A página foi desenvolvida para **apresentação do produto no site oficial da Xiaomi Brasil (MiBrasil)**, seguindo o padrão visual e estrutural utilizado nas páginas de produto da marca.

Atualmente, o **Redmi Pad SE não é mais comercializado no site**, porém a landing page representa um exemplo real de **interface desenvolvida para apresentação de produtos em canais oficiais de tecnologia**.

O objetivo foi criar uma interface moderna, visualmente atrativa e altamente responsiva, destacando os principais recursos do tablet através de **hierarquia visual clara, seções visuais imersivas e storytelling focado nos benefícios do produto**.

A página foi construída com foco em **performance, design responsivo e experiência do usuário**, simulando o padrão utilizado em páginas de produto de grandes marcas de tecnologia.

---

## 🚀 Principais Funcionalidades (Features)

* **Hero Section de Produto:** Apresentação inicial destacando o design e os principais diferenciais do tablet.
* **Storytelling Visual do Produto:** Seções dedicadas que apresentam recursos como tela imersiva, bateria e experiência multimídia.
* **Layout Inspirado em Product Pages de Tech Brands:** Estrutura semelhante às páginas oficiais utilizadas por fabricantes de tecnologia.
* **Design Responsivo:** Interface totalmente adaptável para **Mobile, Tablet e Desktop**.
* **Seções Visuais de Especificações:** Organização clara das características e especificações do produto.
* **Performance Front-end:** Estrutura leve e otimizada sem dependência de frameworks pesados.

---

## 🛠️ Tecnologias Utilizadas

Neste projeto foi adotada uma abordagem **clean e performática**, priorizando carregamento rápido e controle total da interface.

* **HTML5:** Estrutura semântica e organização do conteúdo.
* **CSS3:** Flexbox, CSS Grid, responsividade, animações e variáveis CSS.
* **JavaScript (Vanilla):** Manipulação de DOM e interações da interface.
* **Responsive Design:** Media Queries para adaptação em múltiplos dispositivos.

---

## 🎨 Abordagem de Design (UI/UX)

Como profissional que une **Design e Desenvolvimento Front-end**, o foco desta landing page foi criar uma experiência visual que valorize o produto e facilite a navegação do usuário.

Principais princípios aplicados:

1. **Hierarquia Visual:** Destaque progressivo das funcionalidades do tablet.
2. **Design Orientado à Escaneabilidade:** Conteúdo estruturado para leitura rápida.
3. **Experiência Mobile First:** Interface pensada inicialmente para dispositivos móveis.
4. **Estética Tecnológica:** Visual alinhado com padrões de marcas globais de tecnologia.

---

## 📈 O Impacto (Business Value)

Esta landing page demonstra como **design e front-end podem aumentar a percepção de valor de produtos tecnológicos em canais oficiais e e-commerce**.

Principais benefícios desta abordagem:

1. **Valorização do Produto:** Layout moderno que reforça o posicionamento do dispositivo.
2. **Melhoria da Experiência do Usuário:** Navegação clara e intuitiva.
3. **Engajamento Visual:** Seções visuais que incentivam a exploração do produto.
4. **Padrão de Produto Oficial:** Interface alinhada com a identidade visual utilizada nos canais oficiais da Xiaomi Brasil.
